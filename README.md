# LP.dev - Portfólio Pessoal

Este é o meu portfólio pessoal, desenvolvido como parte da minha jornada de transição para a área de desenvolvimento front-end. O projeto foca em uma interface moderna, responsiva e performática, utilizando React e Tailwind CSS.

## 🚀 Sobre o Projeto
O **LP.dev** foi criado para documentar minha evolução técnica e apresentar meus projetos. Como um aspirante a desenvolvedor front-end de 21 anos, meu objetivo é demonstrar minha capacidade de transformar designs em experiências digitais funcionais e acessíveis.

Atualmente, foco em aprender e aplicar tecnologias modernas para criar soluções eficientes, visando oportunidades como Tech Intern ou Junior Support.

## 🛠 Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

* **React**: Biblioteca principal para a interface do usuário.
* **Tailwind CSS**: Framework para estilização rápida e responsiva.
* **JavaScript (ES6+)**: Lógica e funcionalidade do site.
* **HTML5 & CSS3**: Estrutura e estilização base.
* **Framer Motion**: Adição de animações suaves para uma melhor experiência de usuário.
